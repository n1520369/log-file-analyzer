# Log File Analyzer

## Overview

A Python-based log file analyzer that scans application logs and generates a summary of errors and warnings. This project demonstrates file handling, text processing, and basic log monitoring techniques.

## Features

* Analyze log files
* Count ERROR entries
* Count WARNING entries
* Generate simple log summaries
* Beginner-friendly implementation

## Technologies Used

* Python

## Installation

```bash
python log_analyzer.py
```
